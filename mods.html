Teleport Mod

v0.1.0

A powerful alternative to Cheat Engine for practicing routes and exploring the game world. Save your position anywhere and teleport back instantly with a single key press.

Controls

  • F6 : save current player position and rotation
  • F7 : load saved player position and rotation
IMPORTANT: The readme is made for the Volume Mod but the Teleport Mod is included.
Note: Do not create mods with UE4SS Debugging Tools (OpenGL 3) - External Thread or else it will ask you administrator rights to remove it. Even if you have them you will not be able to remove them. You will have to remove it using the console.
Link Issue? If the Google Drive link doesn't work, you can contact me via discord: cheapmanga_89714

Volume Mod

v0.1.0

See invisible walls and triggers in the game world. Perfect for understanding level boundaries and planning optimized routes for speedrunning.

Features

  • Visualize all invisible walls and collision volumes
  • See trigger zones and interactive areas
  • Essential for route planning and optimization
  • Helps understand level design and boundaries
IMPORTANT: PLEASE READ THE README IN THE .ZIP FOR TUTORIAL
Note: The readme is made for the Volume Mod but the Teleport Mod is also included in the same package.
